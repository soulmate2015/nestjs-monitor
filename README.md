# monitor

## Description

nestjs monitor

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

* filter (alias: f)

* gateway (alias: ga)

* interceptor (alias: i)

* middleware (alias: mi)

* module (alias: mo)

* pipe (alias: pi)

  $ nest generate service users ./modules
